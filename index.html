¿Quién soy?

Consultor Tecnológico Especializado

Soy un consultor tecnológico especializado en análisis de datos, desarrollo web y soluciones digitales. Cuento con experiencia en herramientas como Power BI, Power Query, Excel, SQL Server y MySQL, así como en la creación de páginas y plataformas web.

Además, ofrezco soporte técnico básico en hardware. Me apasiona la tecnología y mi enfoque está en brindar soluciones prácticas, eficientes y orientadas al futuro.

Misión

Brindar soluciones tecnológicas accesibles, eficientes y personalizadas en análisis de datos, desarrollo web y soporte digital, ayudando a personas y empresas a tomar decisiones informadas, optimizar procesos y adaptarse al futuro digital con herramientas innovadoras y confiables.

Visión

Ser una empresa referente en soluciones tecnológicas integrales, reconocida por impulsar la transformación digital de personas y organizaciones mediante el uso estratégico de datos, desarrollo web y soporte tecnológico, contribuyendo a un futuro más conectado, inteligente y accesible.

Portafolio

Análisis de Datos con Power BI

Proyecto de práctica enfocado en el uso de Power BI para el análisis de datos. Se aplicaron técnicas de limpieza, transformación con Power Query y visualización interactiva utilizando diferentes tipos de gráficos y filtros. Ideal para demostrar el flujo completo desde la carga hasta la visualización de datos.

Gestión de Bases de Datos

Manejo consultas y estructuras de bases de datos en MySQL y SQL Server para extraer, organizar y preparar información útil para análisis y visualización.

Desarrollo Web Personalizado

Ejemplo de desarrollo web desde código utilizando Visual Studio Code. Manejo HTML, CSS, JavaScript y herramientas modernas para crear sitios funcionales, responsivos y personalizados.

Plataforma Web Comercial

Plataforma web personalizada para un cliente del sector comercial. El sistema permite registrar movimientos financieros, visualizar balances y calcular el ROI (retorno de inversión) de forma automática. La solución fue creada para facilitar el control financiero del negocio y mejorar la toma de decisiones mediante datos precisos y accesibles desde cualquier dispositivo.
